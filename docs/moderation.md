# Команды модерации

Инструменты для управления сервером и поддержания порядка.

::: callout warning "Права доступа"
Команды модерации требуют соответствующих прав Discord. Убедитесь, что у модераторов есть необходимые разрешения.
:::

---

## Наказания

### `/ban`

Забанить пользователя на сервере.

| Параметр | Описание |
|---|---|
| `пользователь` | Кого забанить (обязательно) |
| `время` | Длительность бана, например `1h`, `1d`, или `0` для перманентного (обязательно) |
| `причина` | Причина бана (необязательно) |

**Требуемые права:** Банить участников (BanMembers)

---

### `/unban`

Снять бан с пользователя по его ID.

| Параметр | Описание |
|---|---|
| `пользователь` | ID пользователя (обязательно) |
| `причина` | Причина разбана (необязательно) |

**Требуемые права:** Банить участников (BanMembers)

::: callout info "Почему ID?"
Забаненные пользователи не видны на сервере, поэтому их нельзя выбрать через меню. Используйте ID пользователя — скопируйте его из логов или через режим разработчика.
:::

---

### `/mute`

Выдать временную блокировку (тайм-аут) пользователю.

| Параметр | Описание |
|---|---|
| `пользователь` | Кого замьютить (обязательно) |
| `время` | Длительность, например `1m`, `1h`, `1d` (обязательно) |
| `причина` | Причина (необязательно) |

**Требуемые права:** Модерировать участников (ModerateMembers)

---

### `/unmute`

Снять тайм-аут с пользователя.

| Параметр | Описание |
|---|---|
| `пользователь` | С кого снять мут (обязательно) |
| `причина` | Причина (необязательно) |

**Требуемые права:** Модерировать участников (ModerateMembers)

---

### `/warn`

Выдать предупреждение (варн) пользователю.

| Параметр | Описание |
|---|---|
| `пользователь` | Кому выдать варн (обязательно) |
| `причина` | Причина (необязательно) |

**Требуемые права:** Кикать участников (KickMembers)

::: callout tip "Авто-наказания"
Настройте автоматические наказания за превышение лимита варнов через `/set autopunish`. Например, при 3 варнах — тайм-аут на 1 час, при 5 — бан.
:::

---

## Управление чатом

### `/clear`

Очистить сообщения в текущем канале.

| Параметр | Описание |
|---|---|
| `число` | Количество сообщений для удаления (обязательно, максимум 100) |

**Требуемые права:** Управлять сообщениями (ManageMessages)

::: callout info "Ограничение Discord"
Discord API позволяет удалять не более 100 сообщений за один раз. Сообщения старше 14 дней не могут быть удалены массово.
:::

---

### `/embed`

Отправить сообщение от имени бота в виде Embed (встроенного сообщения).

| Параметр | Описание |
|---|---|
| `канал` | Канал для отправки (обязательно) |
| `заголовок` | Заголовок Embed (обязательно) |
| `цвет` | Цвет в HTML-коде, например `#FF0000` (обязательно) |
| `текст` | Текст сообщения, до 2000 символов (обязательно) |
| `подпись` | Подпись внизу Embed (обязательно) |

**Требуемые права:** Управлять сообщениями (ManageMessages)

---

## Временные роли

### `/add-timerole`

Выдать временную роль пользователю на определённое время.

| Параметр | Описание |
|---|---|
| `пользователь` | Кому выдать роль (обязательно) |
| `время` | Длительность: `s` (сек), `m` (мин), `h` (час), `d` (дни) (обязательно) |
| `причина` | Причина (необязательно) |

**Требуемые права:** Управлять ролями (ManageRoles)

::: callout warning "Позиция роли"
Роль бота должна быть выше выдаваемой временной роли. Иначе бот не сможет её выдать.
:::

---

### `/set-timerole`

Настройка системы временных ролей.

**Требуемые права:** Управлять ролями (ManageRoles)
