# Команды администрирования

Команды для настройки MercuryBot под ваш сервер. Все команды требуют прав **Администратор**.

::: callout danger "Доступ"
Все команды в этом разделе доступны только пользователям с правом **Администратор** (Administrator) на сервере.
:::

---

## Настройка бота — `/set`

Универсальная команда настройки всех систем бота. Содержит множество подкоманд.

### Каналы

| Подкоманда | Описание |
|---|---|
| `/set notifychannel` | Установить канал для оповещений бота |
| `/set reportchannel` | Установить канал для приёма жалоб |

### Модерация

| Подкоманда | Описание |
|---|---|
| `/set warn` | Установить максимальное количество варнов |
| `/set autopunish` | Настройка авто-наказаний за варны |

**Авто-наказания:**

| Действие | Описание |
|---|---|
| Тайм-аут (1ч) | Мут на 1 час |
| Тайм-аут (24ч) | Мут на 24 часа |
| Кик | Кик с сервера |
| Бан | Бан на сервере |
| Сбросить | Отключить авто-наказания |

### Учётные записи

| Подкоманда | Описание |
|---|---|
| `/set account` | Изменить информацию аккаунта пользователя |

**Доступные настройки:**
- Количество варнов
- Уровень
- XP (опыт)
- Монеты

### Система уровней

| Подкоманда | Описание |
|---|---|
| `/set leveling` | Настройка системы уровней |

**Параметры:**

| Настройка | Описание |
|---|---|
| Игнор канал | Канал, где не начисляется XP |
| Множитель опыта | Множитель для начисляемого XP |
| Максимальный уровень | Лимит уровня |
| XP за сообщения | Количество XP за сообщение |
| XP за войс (в минуту) | Количество XP за минуту в голосе |
| Шаг награды (XP) | Каждые N XP — награда |
| Сумма награды (монеты) | Сколько монет давать за награду |

### Приватные комнаты

| Подкоманда | Описание |
|---|---|
| `/set private` | Настройка приватных каналов |

**Параметры:**
- **Категория** — категория для создания приватных каналов
- **Канал создания приватов** — голосовой канал-триггер
- **Канал управления приватами** — текстовый канал с панелью

### Тикеты

| Подкоманда | Описание |
|---|---|
| `/set ticket` | Настройка системы тикетов |

**Параметры:**

| Настройка | Описание |
|---|---|
| Тикет канал | Канал с панелью создания тикетов |
| Интервал на создание | Кулдаун между тикетами (в минутах) |
| Канал лог тикетов | Канал для логов тикетов |
| Текст сообщения | Текст в панели создания |
| Текст при создании | Текст внутри созданного тикета |
| Роль агента поддержки | Роль, добавляемая в тикет |

### Авто-модератор

| Подкоманда | Описание |
|---|---|
| `/set automoder` | Настройка автомодератора |

**Параметры:**
- **Канал лог** — канал для логов автомодерации
- **Плохое слово** — добавление слова в фильтр
- **Модераторская роль** — роль модераторов
- **Игнорируемый канал** — канал, где автомодерация не работает

### Логирование

| Подкоманда | Описание |
|---|---|
| `/set logs` | Настройки логирования |

**Типы логов:**

| Тип | Описание |
|---|---|
| Действия с ролями | Создание, удаление, изменение ролей |
| Действия с каналами | Создание, удаление, изменение каналов |
| Действия с сообщениями | Редактирование, удаление сообщений |
| Действия с приватными комнатами | Создание, удаление приватных комнат |
| Наказания | Баны, муты, варны, кики |

::: callout tip "Отключение логов"
Чтобы отключить определённый тип логов, укажите `off` вместо канала.
:::

|---|---|

|---|---|

| Подкоманда | Описание |
|---|---|

| Работы | Команда `/work` |
| Работы только в войсе | Таймер работы только в голосе |
| Авто-воркеры | Автоматические работы |
| Уровни | Система XP и рангов |
| Опыт в войсе | Начисление XP за голос |
| Ежедневная статистика в войсе | Статистика голосовой активности |
| Бонусы | Ежедневные бонусы |
| Квесты | Система квестов |
| Достижения | Система достижений |
| Казино | Команда `/casino` |
| Передача валюты (`/pay`) | Переводы между пользователями |
| Трансфер между серверами | Команда `/transfer` |

## Магазин ролей

### `/add-shop-item`

Добавить роль в магазин ролей сервера.

| Параметр | Описание |
|---|---|
| `роль` | Роль для продажи (обязательно) |
| `цена` | Стоимость в монетах (обязательно) |
| `количество` | Лимит покупок (обязательно) |
| `название` | Название в магазине (обязательно) |

### `/remove-shop-item`

Удалить роль из магазина.

| Параметр | Описание |
|---|---|
| `название` | Название предмета в магазине (обязательно) |

## Правила сервера

### `/rules-create`

Создать правило сервера с наказанием.

| Параметр | Описание |
|---|---|
| `наказание` | Название наказания (обязательно) |
| `время` | Длительность наказания (обязательно) |
| `текст` | Текст правила (обязательно) |

### `/rules-edit`

Изменить существующее правило.

| Параметр | Описание |
|---|---|
| `пункт` | Номер пункта правил (обязательно) |
| `параметр` | Что изменить: наказание, время или текст (обязательно) |

### `/rules-delete`

Удалить правило.

| Параметр | Описание |
|---|---|
| `пункт` | Номер пункта правил (обязательно) |

## Розыгрыши

### `/giveaway`

Создать розыгрыш на сервере.

| Параметр | Описание |
|---|---|
| `приз` | Приз розыгрыша, до 60 символов (обязательно) |
| `победители` | Количество победителей, 1–50 (обязательно) |
| `время` | Длительность розыгрыша: `m` (мин), `h` (час), `d` (дни) (обязательно) |

## Панели управления

### `/send-private`

Установить панель управления приватными комнатами в текущем канале.

- Не требует параметров
- Создаёт Embed с кнопкой для создания приватной комнаты

### `/send-loveroom`

Установить панель управления любовными комнатами.

- Не требует параметров

### `/send-ticket`

Установить панель создания тикетов в текущем канале.

- Не требует параметров
- Создаёт Embed с кнопкой для создания тикета

## Быстрый старт настройки

::: callout tip "Рекомендуемый порядок настройки"
1. `/set notifychannel` — канал оповещений
2. `/set logs` — логирование
3. `/set leveling` — система уровней
4. `/set systems` — включить нужные системы
5. `/set private` — приватные комнаты
6. `/set ticket` — тикеты
7. `/send-private` — панель приватов
8. `/send-ticket` — панель тикетов
9. `/add-shop-item` — магазин ролей
:::
