# Основные команды

Команды общего назначения для всех пользователей сервера.

---

## Информация

### `/help`

Информация о доступных командах бота. Показывает список команд, сгруппированный по категориям, с учётом прав пользователя.

- Не требует параметров

### `/ping`

Проверить работоспособность бота и задержку ответа.

- Не требует параметров
- Показывает пинг бота и ответ API Discord

### `/serverinfo`

Подробная информация о текущем сервере.

- Не требует параметров
- Показывает количество участников, каналов, ролей и дату создания

### `/avatar`

Посмотреть аватарку пользователя.

| Параметр | Описание |
|---|---|
| `пользователь` | Чью аватарку показать (обязательно) |

---

## Уровни и ранги

### `/rank`

Проверить уровень и опыт пользователя. Генерирует визуальную карточку ранга.

| Параметр | Описание |
|---|---|
| `пользователь` | Чей ранг проверить (необязательно) |

### `/leaderboard`

Рейтинг пользователей сервера по различным категориям.

| Параметр | Описание |
|---|---|
| `category` | Категория рейтинга (обязательно) |

**Доступные категории:**

| Категория | Описание |
|---|---|
| **по монетах** | Топ по количеству валюты |
| **по уровню** | Топ по уровню и опыту |
| **топ браков** | Топ по длительности браков |
| **по голос. активности** | Топ по времени в голосовых каналах |

---

## Профиль и браки

### `/marry`

Просмотреть статистику брака или сделать предложение пользователю.

| Параметр | Описание |
|---|---|
| `пользователь` | Кому сделать предложение (необязательно) |

::: callout tip "Браки"
Если указать пользователя — бот отправит предложение. Если не указать — покажет вашу текущую статистику брака с карточкой. Брак можно заключить только при наличии достаточной суммы (цена настраивается через `/set marriage`).
:::

## Квесты и достижения

### `/quest`

Меню квестов. Показывает активные квесты, прогресс и награды.

- Не требует параметров
- Доступно до 3 активных квестов одновременно

**Типы наград:**

| Тип | Описание |
|---|---|
| **Уровень** | Повышение уровня |
| **Монеты** | Начисление валюты |
| **Опыт (XP)** | Добавление опыта |

### `/achievments`

Просмотр ваших достижений. Достижения выдаются за различные действия в боте.

- Не требует параметров

## Утилиты

### `/reminder`

Система напоминаний. Создавайте напоминания и просматривайте список.

**Создать напоминание:**

| Параметр | Описание |
|---|---|
| `текст` | О чём напомнить (обязательно) |
| `дата` | Дата в формате ДД.ММ.ГГГГ (обязательно) |
| `время` | Время в формате ЧЧ:ММ (обязательно) |
| `тип` | Где напомнить: **В личные сообщения** или **В текущем канале** (обязательно) |

**Список напоминаний:**

```
/reminder list
```

::: callout info "Часовой пояс"
Напоминания работают по времени GMT+3 (Москва). Указывайте время соответственно вашему часовому поясу.
:::

---

### `/coin`

Игра в орла и решку. Угадайте сторону монеты.

| Параметр | Описание |
|---|---|
| `side` | Выберите: **орёл** или **решка** (обязательно) |

## Обратная связь

### `/bug`

Сообщить об ошибке или баге в работе бота. Открывает модальное окно для ввода описания проблемы.

- Не требует параметров

### `/idea`

Отправить предложение или идею для улучшения бота. Открывает модальное окно для ввода идеи.

- Не требует параметров

### `/report`

Написать жалобу на пользователя сервера.

| Параметр | Описание |
|---|---|
| `пользователь` | На кого жалоба (обязательно) |
| `текст` | Текст жалобы (обязательно) |

::: callout warning "Канал жалоб"
Для работы команды `/report` администратор должен настроить канал через `/set reportchannel`.
:::

## Конфигурация (только для администраторов)

### `/config`

Просмотр конфигурации сервера в удобном меню. Доступны разделы:

- **Основное** — общие настройки
- **Каналы** — настройка каналов и категорий
- **Экономика** — настройка экономики и наград
- **Тикеты** — настройка системы тикетов
- **Системы** — включение/выключение систем бота

::: callout danger "Требуются права"
Команда доступна только пользователям с правами **Администратор**.
:::
